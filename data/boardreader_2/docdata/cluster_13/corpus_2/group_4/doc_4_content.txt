 Features:      - Skinable, All controls and containers use skinned background, skinned highlight background, skinned text color.      - Provides stock controls which is lacked on S60, like checkbox, choicelist and so on.      - Two methods of construction, from code or from resource      - Auto-Layout and auto focus-managing. Either dialog or view uses a a container, which layout childeren control row by row. Row height can be specified to minimum height, actual height,fixed height or grab all available space.      - Resource definition can be switched easily between dialog and view. They use the same container resource definition.      - Both dialog and view can be scollable, can have multiple pages, and support CBA and menus.      - Derivation is easy. Developers have to write only a few lines to implement dialogs and views. They do not have to care about the managing of focus and layout.
